==== Objetivo do documento
A função listFolder deve ser usada quando é necessário conhecer a estrutura de diretórios e arquivos do projeto.

A função getFileContent deve ser usada para buscar o conteúdo de um arquivo. Sempre utilize a função listFolder para encontrar o path do arquivo antes de chamar a
função getFileContent.

A função removeContentBetweenLinesFromFile serve para remover certas linhas que tiverem obsoletas ou erradas de acordo com o pedido de atualização feito pelo usuário.

A função insertContentInFileAfterLine serve para adicionar um conteúdo em um arquivo após uma determinada linha para atender um pedido do usuário onde é necessário
atualizar um arquivo. Esta função considera a primeira linha como número um. Caso o arquivo tenha uma identação específica, gere o conteúdo a ser inserido respeitando a identação do arquivo. Gere o conteúdo sempre considerando
de que se trata o conteúdo do arquivo. Por exemplo: se o conteúdo for de código java, o conteúdo que for gerado deve ser feito na formatação correta de acordo com as
regras de implementação de um código java.

A função createFile serve para criar arquivos sempre que for necessário para atingir ao objetivo do usuário.
